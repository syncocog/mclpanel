# MCLEGENDS.GG Features

## Core Features (Pterodactyl Parity)

### Server Management
- Create, start, stop, restart, suspend servers
- Resource allocation (CPU, RAM, disk)
- Docker-based isolation
- Multi-server support
- Server templates

### Real-time Console
- Live console output streaming
- Command execution
- Console history
- WebSocket-powered updates

### User Management
- Multi-role system (admin, user)
- JWT authentication
- 2FA support
- OAuth2 integration (Google, Discord)
- Activity logging

### File Management
- Upload/download files
- Edit configuration files
- Mod/plugin uploads
- Directory browsing

### Scheduling
- Automated backups
- Scheduled restarts
- Custom cron jobs
- Task management

## Minecraft-Specific Features

### Player Management
- Whitelist GUI
- Ban/unban interface
- Kick players
- Player activity tracking

### Server Configuration
- Modpack installer
- Plugin manager
- Version selector
- Server type selection (Spigot, Paper, Forge, Fabric)

### Analytics
- TPS monitoring
- Player count tracking
- Latency metrics
- Performance graphs

### Backups
- Automatic snapshots
- Manual backups
- Restore functionality
- S3-compatible storage

## Futuristic UI

- Neon theme (blue, purple, pink, green)

- 3D performance graphs
